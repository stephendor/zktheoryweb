# Archived: Morse-Smale / IMD Geographic Paper

**Archived:** 2026-04-02
**Reason:** This draft predates the 10-paper programme. It uses poverty_tda domain
data (IMD 7D deprivation scores for UK LSOAs) rather than the trajectory_tda
embedding that the programme defines for P04. The IMD data lacks sufficient
dimensionality to produce meaningful higher-order topological output.

## Contents

- `v1-draft.md` — ~10,000-word Morse-Smale paper on IMD geographic deprivation
- `outline.md` — Original paper outline
- `supplementary_outline.md` — Supplementary material outline

## Potential Reuse

A Morse-Smale approach could be revisited if:
- A higher-dimensional poverty surface is constructed (e.g., combining multiple
  census rounds, or using panel-level income trajectories as the surface)
- TTK/ParaView integration is mature enough for production use
- A standalone poverty_tda methodology paper is pursued outside the 10-paper programme

These materials are not deleted — they represent genuine analytical work that
may inform future directions.
