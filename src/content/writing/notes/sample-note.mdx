This is a rough working note, not a finished argument.

The zero-th Betti number $\beta_0$ of a Vietoris–Rips complex built on a deprivation dataset counts the number of connected components. In a fully connected deprivation landscape, $\beta_0 = 1$. High values suggest isolated pockets of poverty that are not "adjacent" in any welfare-space sense to the mainstream poor.

Could $\beta_1$ — the count of one-dimensional holes, i.e., loops — represent something meaningful about the cyclical structure of deprivation traps? I genuinely do not know yet. The intuition is appealing; the interpretation is unclear. Flagging for later.

Key reference to chase up: Carlsson (2009), "Topology and Data", *Bulletin of the AMS*.
