## When Is a Loop Real?

A persistence diagram is a mathematical object, not a statistical test. The stability theorem guarantees that small perturbations to the point cloud produce small changes in the diagram — but it does not tell you whether the observed topology exceeds what you would expect from pure sampling noise. A dataset of 500 households randomly sampled from a uniform distribution will still produce a persistence diagram with non-zero H₁ features; some finite subset of 500 random points will, by chance, form near-ring configurations that generate apparent loops. The question every applied TDA analysis must answer is not "is there topology in my diagram?" but "is the topology I observe in excess of what I would expect from noise?" That is a statistical question, and it requires statistical tools.

Two primary approaches exist: **bootstrap confidence band testing** and **permutation testing**. They test different null hypotheses and are appropriate in different analytic situations. Both are implemented in `persim` and through standard Python statistical libraries. Neither requires a parametric assumption about the data distribution.

## Bootstrap Confidence Bands

The bootstrap test treats the observed point cloud as a finite sample and asks whether the observed topological features are stable across different subsamplings of the same population. The procedure:

1. Draw $B$ bootstrap samples $X^{(1)}, \ldots, X^{(B)}$, each of size $n$ (sampling with replacement from $X$).
2. For each bootstrap sample, compute the persistence diagram $\text{Dgm}^{(b)}$.
3. For each diagram, record the maximum H₁ persistence value $p^{(b)} = \max_{(b,d) \in \text{Dgm}^{(b)}_1} (d - b)$.
4. The bootstrap null distribution is $\{p^{(b)}\}_{b=1}^B$.
5. A confidence threshold at level $\alpha$ is the $(1-\alpha)$ quantile $\hat{q}_{1-\alpha}$ of this distribution.
6. The observed feature is significant at level $\alpha$ if $p_{\text{obs}} > \hat{q}_{1-\alpha}$.

This test is directly interpretable: it asks whether the observed loop survives the sampling perturbation. If 95% of bootstrap subsamples produce a maximum H₁ persistence below the observed value, the feature is unlikely to be a pure sampling artefact.

In practice, $B = 500$ iterations is sufficient for an exploratory test; $B = 2{,}000$ is recommended for publication-grade results. Computation time scales linearly with $B$ and quadratically with $n$, so for large datasets, run the bootstrap over landmark-subsampled data (Module 2).

## Permutation Tests

The permutation test addresses a different null hypothesis: not "could this topology arise from noise?" but "does this topology depend on the specific labelling or grouping structure I am investigating?" For poverty data, a common application is to test whether the topology of welfare trajectories in one region or demographic group differs from another. The null hypothesis is that group labels are arbitrary — the observed topological difference is no larger than what would arise if households were randomly reassigned between groups.

Procedure for a two-sample permutation test:

1. Compute observed diagrams $\text{Dgm}_A$ and $\text{Dgm}_B$ for groups $A$ and $B$.
2. Compute observed test statistic $T_{\text{obs}} = d_W(\text{Dgm}_A, \text{Dgm}_B)$ using Wasserstein-1 distance.
3. Pool the data: $X_{\text{pool}} = X_A \cup X_B$.
4. For each of $B$ permutations: randomly assign $|X_A|$ observations to group $A'$ and $|X_B|$ to group $B'$; compute $T^{(b)} = d_W(\text{Dgm}_{A'}, \text{Dgm}_{B'})$.
5. P-value $= \frac{1}{B} \sum_b \mathbf{1}[T^{(b)} \geq T_{\text{obs}}]$.

The Wasserstein-$p$ distance used as the test statistic is:

$$W_p(X, Y) = \left( \inf_\gamma \sum_{x \in \text{Dgm}(X)} \| x - \gamma(x) \|^p \right)^{1/p}$$

where the infimum is over all matchings $\gamma$ between points of the two diagrams (extended to the diagonal), and the norm is the $L^p$ norm on the plane. With $p = 1$ (Wasserstein-1), the test is more robust to outliers; with $p = 2$ (Wasserstein-2), large persistent features dominate the distance and the test is more sensitive to changes in the strongest structural features.

## A Worked Poverty-Data Example

Consider a national welfare survey with households grouped into two cohorts: those who received Discretionary Housing Payments (DHP) during a benefit cap period (Cohort A, $n_A = 1{,}843$) and those who did not (Cohort B, $n_B = 2{,}107$). Both cohorts are observed across six annual waves and represented as normalised welfare vectors in a shared six-dimensional space. The research question: does the topological structure of poverty trajectories differ between DHP recipients and the comparison group?

Compute H₁ diagrams for both cohorts separately. The observed Wasserstein-1 distance between the diagrams is $T_{\text{obs}} = 0.42$. Run 1,000 permutations, randomising cohort assignment while preserving group sizes. The null distribution of $T^{(b)}$ has mean 0.18 and 99th percentile 0.31. The observed distance 0.42 exceeds all 1,000 permuted values: one-tailed p-value $< 0.001$.

The topological conclusion: the H₁ structure of welfare trajectories differs significantly between DHP and non-DHP households — the cyclical poverty dynamics visible in their trajectory data are not drawn from the same topological population. This finding motivates substantive analysis of *which* features differ, using the Persistence Diagram Builder (Module 4) to directly compare the diagrams.
