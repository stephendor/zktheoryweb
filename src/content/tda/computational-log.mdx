# Computational Log

This page serves as a public computational laboratory notebook, documenting hardware, runtimes, cloud resource decisions, and reproducibility notes for each paper in the TDA research programme. The goal is full computational transparency: every result in every paper should be reproducible given the recorded environment and inputs.

---

## Entry Format

Each log entry records: date, paper reference, task description, hardware used, approximate runtime, output files, and any environment notes.
