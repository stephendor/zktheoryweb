# Visualisations

This gallery collects the interactive and static visualisations produced across the ten-paper TDA research programme. Interactive components are built as self-contained React islands using D3.js; static figures are exported as SVG/PDF from the paper analysis pipelines.

---

## Planned Interactives

### Persistence Diagram Explorer *(Paper 1)*

An interactive barcode diagram viewer allowing exploration of the MML persistence output across demographic sub-groups. Users will be able to toggle between male and female subsamples, adjust the significance threshold, and inspect individual trajectory examples corresponding to selected topological features.

**Status:** Planned — pending Paper 1 final dataset.

---

### Mapper Graph Viewer *(Paper 2)*

An interactive force-directed Mapper graph displaying the five topological clusters of UK employment trajectory space. Nodes are coloured by cluster membership; clicking a node reveals the representative career sequences in that group. Lens parameter sliders allow exploration of parameter sensitivity.

**Status:** Planned — pending Paper 2 analysis completion.

---

### Zigzag Complexity Timeline *(Paper 3)*

An animated timeline showing the evolution of employment trajectory topology from 1991 to present, with highlighted recession periods. The topological complexity index is plotted alongside official unemployment rates to illustrate the leading-indicator property.

**Status:** Planned.

---

### Welfare State Topology Map *(Paper 5)*

A choropleth map of Europe with countries coloured by H₁ persistence entropy, enabling visual comparison of employment trajectory topology across welfare state regimes.

**Status:** Planned — pending EU-SILC data access.
